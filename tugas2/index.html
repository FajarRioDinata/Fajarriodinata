01. Pengertian OOP

Object Oriented Programming (OOP) Object Oriented Programming (OOP) adalah paradigma pemrograman yang berorientasi pada objek, dimana program dibuat menggunakan class dan objek. Dalam OOP terdapat beberapa konsep utama seperti Encapsulation, Inheritance, Polymorphism, dan Abstraction yang membantu membuat program lebih terstruktur, modular, dan mudah dikembangkan.

02. Konsep Dasar

  • Class: Blueprint atau cetakan untuk membuat objek.
  • Object: Instance nyata yang tercipta dari sebuah class.
  • Attribute: Representasi data melalui variabel di dalam class.
  • Method: Logika fungsional atau fungsi di dalam class.

03. Class Sederhana (Kapal)

Pada contoh program di bawah ini terdapat class Kapal yang memiliki atribut merk dan tipe untuk menyimpan spesifikasi armada laut. Objek ini dilengkapi dengan method berlayar() dan buangJangkar() guna merepresentasikan operasional kapal.

DART SOURCE CODE

// Nim    : 251410025
// Kelas  : SI2A

class Kapal {
  String? merk;
  String? tipe;
  
  void berlayar() {
    print('Kapal $merk sedang membelah lautan...');
  }
  
  void buangJangkar() {

  kapalFajar.buangJangkar();
}

07. Encapsulation

Encapsulation di sini digunakan pada class Toko untuk melindungi variabel privat _stokBarang. Data stok hanya bisa ditambah melalui method pasok() guna menjamin validitas jumlah barang di gudang.

DART SOURCE CODE
// Nama   : Fajar rio dinata
// Nim    : 251410025
// Kelas  : SI2A

class Toko {
  String namaToko;
  int _stokBarang; 

  Toko(this.namaToko, this._stokBarang);

  void pasok(int jumlah) {
    if (jumlah > 0) _stokBarang += jumlah;
  }
  
  int get lihatStok => _stokBarang;
}

void main() {
  Toko tokoFajar = Toko("Fajar Electronic", 45);
  print('Unit Usaha: ${tokoFajar.namaToko}'); 
  tokoFajar.pasok(15);
  print('Stok Terkini: ${tokoFajar.lihatStok} unit'); 
}

09. Product Class (Fisik)

Class ProdukFisik menerapkan "Named Parameters" dalam constructor untuk menghindari kesalahan pengisian argument. Metode ini mempermudah identifikasi data seperti barcode, nama, dan berat saat objek diciptakan.

DART SOURCE CODE
// Nama   : Fajar rio dinata
// Nim    : 251410025
// Kelas  : SI2A

class ProdukFisik {
  final String barcode, nama; 
  final double berat; 

  ProdukFisik({
    required this.barcode,
    required this.nama,
    required this.berat,
  });
}

void main() {
  ProdukFisik barang1 = ProdukFisik(
    barcode: "888-001",
    nama: "Monitor UltraWide",
    berat: 4.5,
  );

  ProdukFisik barang2 = ProdukFisik(
    barcode: "888-002",
    nama: "Mechanical Mouse",
    berat: 0.15,
  );

  print('Barang 1: ${barang1.nama} (Berat: ${barang1.berat}kg)'); 
  print('Barang 2: ${barang2.nama}'); 
}
